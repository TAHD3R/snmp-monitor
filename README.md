# 机房温湿度检测脚本

## 介绍

该脚本旨在提供关于服务器机房温湿度监测的反馈。

## 功能

- 从传感器中获取温湿度数据
- 分析数据以识别异常或趋势

## 安装

1. 克隆仓库：`git clone https://github.com/tahd3r/snmp-monitor.git`
2. 安装所需依赖：`poetry install`
3. 运行脚本：`python main.py`

## Docker安装

1. 编排镜像：`docker-compose up -d`

## 使用

- 脚本将自动按照设定的时间间隔收集温湿度数据。
- 您可以在 `.env` 文件中自定义时间间隔和其他设置。
- 脚本将基于配置的阈值发送通知。
